How to install:

  • Install the game
  • Install the 3.9.68 patch
  • Install the Data Update
  • Copy over the files from my patch, replacing everything
  • Right-click the executable (cm0102.exe), go to Properties. In the Compatibility tab, change it to wither Windows 95 (if you plan on playing network games) or Windows XP Service Pack 3

Note: Some antivirus programs may flag either cm0102.exe or NickDLL.dll as malware. There's nothing I can do about that unfortunately, and Nick+Co has mentioned that it is simply a false positive.
